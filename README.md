## Description

The objective is to learn modern vanilla JavaScript by implementing professional-grade web application development methodologies & toolings:

- User stories
- Wireframing
- Data modeling
- Linting
- Code formatting
- MVP
- Build & deployment
- CI/CD
- E2E testing
